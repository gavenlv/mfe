# ShopMFE 文档导读

## 怎么用这套文档

**如果你是完全的小白**：按 01 → 07 顺序读。读到不懂的名词，先去 07 查，再回来继续。

**如果你有前端基础但不懂微前端**：直接读 02、03、05，然后挑 06 里感兴趣的模块看代码。

**如果你只想跑起来**：看根目录 [README.md](../README.md) 的"快速开始"。

```
        浏览器 http://localhost:5173
                │
        ┌───────▼────────┐
        │  Shell 主应用    │  路由 / 布局 / 购物车角标
        └─┬───┬───┬───┬──┘  （通过 Module Federation 动态加载下面 5 个微前端）
          │   │   │   │
```

## 配套代码说明

所有源码都已加上**教学型中文注释**。读代码时你会看到三类注释：

- **文件级注释**（文件顶部 `/** ... */`）：这个文件是什么、属于哪个模块、关键概念
- **概念注释**（多行）：遇到重要原理时展开讲"为什么"
- **行内注释**（`//`）：解释具体这一行在做什么

建议读代码顺序：`shared/` → `server/` → `shell/` → 4 个微前端。
